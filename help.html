General
Generate
The generate button uses the URL entered in the URL textfield and generates a screenshot from that page. This works with an external server that is hosted for free so it is ideling when no one uses it. Therefore it can take some time until a screenshot is generated.
Browse..
This button lets you select an image from your computer and will upload it automatically on imgur. So DON'T use private images, even if the link for the image is not public.
